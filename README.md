# 👗 Regal Thread – Fashion E-commerce Website

A PHP + MySQL-based fashion e-commerce front-end for showcasing modern clothing products.

## 🎯 Features
- Product gallery with images
- Shopping cart UI (non-functional)
- Organized static pages
- SQL file for database structure

## 🛠️ Tech Stack
- PHP
- MySQL
- HTML/CSS
- JavaScript

## 🚀 How to Run
1. Clone the repo  
   `git clone https://github.com/Shreecant/Regal-Thread.git`

2. Import `fashion.sql` into your local MySQL server

3. Deploy files to XAMPP/LAMP `htdocs` folder

4. Visit `http://localhost/Regal-Thread/` in browser
